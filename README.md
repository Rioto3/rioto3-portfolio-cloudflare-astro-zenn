## セットアップ

### 必要な環境
- Node.js 18.x
- pnpm

### インストール手順
```bash
git clone https://github.com/Rioto3/rioto3-portfolio-cloudflare-astro-zenn
cd rioto3-portfolio-cloudflare-astro-zenn
pnpm install
pnpm run dev
```
